# Audi S4 — GitHub Pages Listing

A premium car-dealership-style single-page website showcasing a **2014 Audi S4** with the **7-speed S tronic DSG** dual-clutch transmission and Quattro all-wheel drive.

Inspired by the [Autohaus Framer template](https://www.framer.com/marketplace/templates/autohaus/), this static GitHub Pages site features:

- **Hero section** — full-screen background image, year / model headline, live stat counters (HP, 0–60, top speed, gearbox)
- **Highlights** — three feature pillars (engine, DSG, Quattro)
- **Vehicle Overview** — description with floating model-year badge and checklist
- **Full Specifications** — tabbed spec sheet (Engine · Drivetrain · Dimensions · Performance)
- **Photo Gallery** — 5-image masonry grid with lightbox viewer and keyboard navigation
- **Features Accordion** — six S4 standard-equipment items (seats, MMI, B&O audio, Drive Select…)
- **Inquiry Form** — contact form with success state
- **Responsive layout** — mobile-first, works on all screen sizes
- **Dark premium aesthetic** — Audi red accent colour, Montserrat + Playfair Display typography

## Structure

```
index.html          Main landing page
css/style.css       All styles (dark theme, responsive)
js/main.js          Navbar, tabs, gallery lightbox, form, scroll animations
```

## Local Preview

Open `index.html` in any modern browser — no build step required.

## GitHub Pages

Enable GitHub Pages from **Settings → Pages → Deploy from branch → main / (root)** and the site will be live at `https://<username>.github.io/audi_s4/`.
